🔢 Calculadora de Controle Logístico

Sistema simples de controle de porcentagem desenvolvido em **HTML, CSS e JavaScript puro**.

🚀 Funcionalidades

- Cálculo de porcentagem
- Modo descarregamento e carregamento
- Barra visual de progresso dinâmica
- Calculadora integrada para apoio operacional
- Sistema de anotações com salvamento automático (localStorage)
- Botão para copiar resultado

🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

🎯 Objetivo

Este projeto é uma ferramenta de apoio operacional utilizada em ambiente logístico para acompanhamento de processamento em tempo real.
